UCI Retail II Dashboard

Microsoft Power BI, Power Query, Dax

Dashboard


In this dashboard the following metrics were investigated: sales KPIs, revenue, average customer spend and orders. Additionally the top ranked countries and products by revenue were examined. This was designed as an initial business overview that would help stakeholders understand their overall performance both throughout the year, and relative to the previous year. From this clients could develop further lines of inquiry with the analyst, leading to actionable insights that drive growth.
All data used in the model was collected from the Kaggle website.

Data Model


The data was modelled using a star schema which included 4 dimension tables created to both normalise the fact table, and allow for efficient grouping and filtering.

A date dimension table was defined using the calendar function to ensure there was a continuous date range for the time intelligence calculation group.

Additionally, a table was added for storing and organising measures.

DAX Formula Examples


Revenue

Average Spend

Average Spend = AVERAGEX(VALUES(Sales[Customer ID]),[Revenue])

Time Intelligence Measures


Year to Date

YTD = CALCULATE(SELECTEDMEASURE(), DATESYTD('Date Table'[Date]))

Last Year to Date

LYTD =
VAR max_date = MAX('Date Table'[Date])
VAR max_ly = EDATE(max_date,-12)
VAR lytd_dates = FILTER(SAMEPERIODLASTYEAR(DATESYTD('Date Table'[Date])),'Date Table'[Date]<=max_ly)

RETURN CALCULATE(SELECTEDMEASURE(),lytd_dates)

Year-over-Year

YOY (%) =
VAR CY = CALCULATE(SELECTEDMEASURE(),'Time Intelligence'[Calculation group column] = "YTD")
VAR LY = CALCULATE(SELECTEDMEASURE(),'Time Intelligence'[Calculation group column] = "LYTD")
RETURN DIVIDE(CY-LY,LY)
